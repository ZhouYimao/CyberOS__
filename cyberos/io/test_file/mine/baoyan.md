# 关于高水平科技竞赛获奖本科生推免加分的规定

# 说明：  

如同一名同学参加多个竞赛并获奖，加分以最高值计，不累加； 依据本规定所得加分与从学校相关负责单位获得的加分以最高值计，不累加； 本规定适用于2019 级本科生，由学院推免遴选工作小组负责解释。
